    (一).   前提是已經在 /usr/bin/ 中建立 arm-none-eabi-gcc 的 soft link
    (二).   直接在 shell 打指令 make 就會把 hw03.s 編譯成 hw03.exe 
    (三).   在 shell 打 make clean 就會把 hw03.exe 刪除

2.  解題邏輯
    (一)    矩陣 A = 2 * 4 matrix, B = 4 * 2 matrix, C = 2 * 2 matrix
            (i).    A 、 B 各有8個變數， C 有 4 個
            (ii).   依序讀取 a(1,1)到a(1,4)，並存放在 r1 到 r4 內。
                    依序讀取 b(1,1)到a(4,1)，並存放在 r6 到 r9 內。
                    將 a(1,1) * b(1,1) 之結果存放在 r10。
                    將 a(1,2) * b(2,1) 之結果存放在 r6。
                    將 a(1,3) * b(3,1) 之結果存放在 r7。
                    將 a(1,4) * b(4,1) 之結果存放在 r8。
                    將上述四個結果之值加總並存放在 r10。
                    最後將 r0 之值存放到 c(1,1)。
                    之後重複以上邏輯，只是 index 會有所不同。

                    c(1,1) = a(1,) * b(,1)
                    c(1,2) = a(1,) * b(,2)
                    c(2,1) = a(2,) * b(,1)
                    c(2,2) = a(2,) * b(,2)
